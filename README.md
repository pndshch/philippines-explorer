# フィリピン・レンズ

フィリピンについて詳しくなるための静的ブラウザアプリです。

- 島グループ別の地図
- Three.js による3D探検ゲーム
- 18地域の整理
- 6つの学習視点
- Filipino フレーズカード
- 歴史タイムライン
- 10問クイズ

GitHub Pages でそのまま公開できるよう、ビルド不要の静的ファイルで構成しています。
3D描画には CDN 版 Three.js を使っています。
3D探検ゲームは、SRTM / Natural Earth / GADM / OpenStreetMap などの公開データで本格化できる前提を説明しつつ、ページを軽く保つため学習用の簡略化モデルとして実装しています。
